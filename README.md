# Expo Skeleton

React Native app with Expo 54, NativeWind 4, and expo-router.

## Setup

```bash
bun install
```

## Development

```bash
bun start
```

Scan the QR code with Expo Go app.

## Pages

- `/` - Dashboard with navigation
- `/settings` - Theme toggle (persisted)

## Stack

- Expo 54 + expo-router 6
- NativeWind 4 (Tailwind CSS)
- AsyncStorage for theme persistence
- TypeScript

## Assets

Add these placeholder images to `/assets`:

- `icon.png` (1024x1024)
- `splash-icon.png` (512x512)
- `adaptive-icon.png` (1024x1024)
- `favicon.png` (48x48)

Or run without them - Expo will use defaults.
